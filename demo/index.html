Steganography hides bytes from the eye, not from statistical analysis. Paste any text here to see whether it carries hidden zero-width chars or unexpected trailing whitespace, and how many payload bytes that implies. No password required - detection is not the same as decryption, but it reveals that a hidden message exists.
How detection works

zero-width method: the encoder inserts U+200B and U+200C characters into the text. These chars never appear in normal prose. Counting them immediately reveals the number of bits, and dividing by 8 gives the hidden byte count. The first 32 bits are the length header, so you can read the declared payload size without ever decrypting.

whitespace method: trailing tabs are extremely rare in normal text. A high count of trailing tabs across many lines is strong evidence of a hidden payload. The decoder reads them as bits to get the length header.

Both methods are designed to resist casual visual inspection, not statistical steganalysis. This is a fundamental property of LSB-style steganography: it hides content from humans, not from analysis tools.
